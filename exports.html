How It Works
① You paste your Apps Script Web App URL below

③ Apps Script emails it via your Gmail for free

⚙️ Google Apps Script Setup
Step 1 · Go to script.google.com
Step 2 · Create new project, paste the GAS code
Step 3 · Deploy → Web App → Anyone → Copy URL
Step 4 · Paste URL + choose a Secret Token below
Free limits: 100 emails/day (Gmail) · 1,500/day (Workspace)
No API keys · No account creation · Uses your Google account
